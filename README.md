# E-Commerce Tech

A modern full-stack e-commerce web application built with Next.js, featuring product browsing, search, filtering, authentication, shopping cart, mock checkout, order history, and an admin dashboard for product management.

## Features

- Responsive homepage
- Product catalog
- Product detail page
- Product search
- Category and price filters
- Product sorting
- Shopping cart
- Login and registration
- Mock checkout
- Order history
- Admin product management
- Responsive desktop and mobile design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
- GitHub
- Vercel
